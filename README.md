🩺 Health Checker Web App

A simple and interactive web application that analyzes user symptoms and provides possible illness suggestions, severity levels, and basic remedies.

🚀 Features
Symptom-based health analysis
Severity level indicator (Mild, Moderate, High)
Medicine suggestions
Dark mode support
Loading animation
LocalStorage support
🛠 Tech Stack
HTML
CSS
JavaScript
⚠ Disclaimer

This project is for educational purposes only and does not provide medical advice.
